# 🚀 Ranking Report & Advanced Caching System

To optimize AI response times and significantly reduce token costs, our platform utilizes an **Advanced Message Ranking** and **Multi-layer Caching System**. This interface gives you full control over your AI agents' memory and performance.

---

### 📊 1. Core Performance Metrics
The top dashboard cards display the real-time efficiency of your caching mechanism:

*   **🎯 Cache Hit Rate (%)**: The percentage of queries resolved directly from the cache without calling the AI model.
*   **✅ Total Hits**: The number of times the system successfully served a cached response (indicating direct token savings).
*   **❌ Cache Miss**: Queries that were not found in the cache and required a new AI request.
*   **📈 Total Queries**: The cumulative number of messages or requests processed by your system.

---

### 🔝 2. Message Ranking & Control
This section lists your most frequent queries, ranked by their impact:

*   **🔄 Frequency**: Tracks how many times a specific message has been asked.
*   **💰 Tokens Saved**: Displays the exact amount of tokens saved by serving the response from the cache.
*   **🗑️ Action (Delete)**: If an incorrect or outdated response is cached, you can remove it instantly using the **Delete** icon.
*   **🔍 Filtering**: Filter by specific AI Agents to view individualized caching reports.

---

### 🤝 3. Smart Cache Sharing & Privacy
A unique feature that allows experience sharing between different AI agents:

*   **🤖 Active Agent Filtering**: Select the agent you wish to optimize.
*   **📤 Share Cache From**: Import cached responses from other agents to broaden the current agent's knowledge base.
*   **🏷️ Labels (Shared/Blocked)**: Messages imported from other agents are marked as **"Shared"**. If you choose to restrict an imported response, it will appear as **"Shared Blocked"**.
*   **🔒 Privacy Lock**: Use the **Lock** icon next to any message to prevent it from being shared with other agents, ensuring strict data privacy.

---

### ⭐ 4. Request Premium Cache (Paid Feature)
For enterprise-level stability, we offer a **Premium Caching** tier:

*   **🏛️ Permanent Storage**: Unlike standard cache, Premium data is archived long-term (years) without expiration.
*   **📌 Static Data Optimization**: Best for unchanging information like company policies or FAQs.
*   **💹 Cost Efficiency**: While it involves a monthly fee, it drastically reduces long-term token consumption, making it highly profitable for scaling businesses.

---

### 🚫 5. General User Restrictions
To maintain system integrity and security for all users:

> [!WARNING]
> **Global Cache Protection**: Standard users cannot clear the Global Cache (the option may be visible but restricted at the system level).

> [!CAUTION]
> **Cache Type Rigidity**: Users cannot manually toggle between Global Sync and Agent Specific cache types.
