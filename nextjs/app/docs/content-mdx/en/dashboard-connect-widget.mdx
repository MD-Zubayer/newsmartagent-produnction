# 🌐 Web Widget: Your Smart On-Site Assistant

Transform your website into a 24/7 conversion engine with the NSA Web Widget. Create multiple custom chat widgets and integrate them into any site by simply pasting a few lines of code before the `</body>` tag.

---

### 🎨 1. Full Appearance Customization

Tailor the widget to reflect your brand identity perfectly with these features:

*   **🌈 Colors & Themes**: Set your primary brand color using hex codes to match your website's palette.
*   **🖼️ Icon Branding**: Upload your custom brand logo or use an image URL as the widget icon.
*   **📐 Size & Shape**: Adjust the size (Small to Large) and corners (square to round) to match your UI.
*   **📍 Flexible Positioning**: Place the widget in any of the four corners (Bottom/Top, Right/Left).
*   **💭 Bubble Control**: Choose whether to show or hide the chat bubble background for a cleaner look.

---

### 🤝 2. Multi-Channel Support

The widget acts as a powerful bridge to all your connected communication channels:

*   **📱 WhatsApp & Messenger**: Link your connected AI agents directly to the widget for a seamless experience.
*   **🔗 Manual Overrides**: Add custom phone numbers or social links for specific departments or agents.

---

### ⚙️ 3. Effortless Integration

Once customized, the system generates a lightweight **Integration Manifest** snippet:

1.  **📋 Copy**: Grab the auto-generated JS snippet from your dashboard.
2.  **📌 Paste**: Add it to your website's HTML just before the `</body>` tag.

---

### 🚀 Platform Setup Guide

| Platform | Where to Paste Code |
| :--- | :--- |
| **WordPress** | Use "WPCode" plugin or edit `footer.php` before `</body>`. |
| **Shopify** | Edit `theme.liquid` and paste before `</body>`. |
| **Next.js** | Use `next/script` in `layout.js` or `_document.js`. |
| **Wix / Sq.Space** | Settings > Custom Code > Body End / Footer section. |

---

> [!TIP]
> **Developer Tip**: The widget is highly dynamic. Once the code is installed, any design or logic changes you make in the NSA dashboard will update on your website automatically—no redeployment required!
