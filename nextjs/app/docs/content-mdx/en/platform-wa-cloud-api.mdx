# WhatsApp Cloud API vs. Linked Devices

NewsSmartAgent primarily uses the **Linked Devices (QR Scan)** method for its ease of use, but we also support the official **WhatsApp Cloud API** for enterprise-grade stability.

### 🌓 Comparison

| Feature | Linked Devices (QR) | Cloud API (Official) |
| :--- | :--- | :--- |
| **Ease of Setup** | ⚡ Very Easy | 🛠️ Complex |
| **Stability** | 📱 Needs Phone Online | 🌐 Always Online |
| **Cost** | 🆓 Free (NSA Only) | 💳 Per Message (Meta) |
| **Broadcasts** | 🚫 Limited | ✅ Unrestricted |

### 🛠️ Switching to Cloud API

1.  Provision a **Meta Developer Account**.
2.  Obtain your **Permanent Token** and **Phone Number ID**.
3.  Enter these details in **Platforms > Advanced WhatsApp Settings**.

> [!WARNING]
> Switching to Cloud API requires a separate business verification from Meta. Ensure your legal documents are ready.
