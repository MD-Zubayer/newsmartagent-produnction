# API Key Management

For developers and advanced users, NewsSmartAgent provides direct API access to integrate our AI features with your own external applications.

### 🔑 Generating Keys

1.  Navigate to **Settings > Developer Portal**.
2.  Click **'Generate API Key'**.
3.  Give the key a descriptive name (e.g., 'My_External_App').
4.  Copy and save the key immediately. It will not be shown again for security reasons.

### 🛡️ Rotation and Revocation

*   **Rotate Key**: Invalidate the old key and generate a new one if you suspect a leak.
*   **Revoke**: Instantly disable any key to stop external access.

> [!CAUTION]
> Your API Key allows full access to your account data and balance. **Never** push your keys to public repositories like GitHub.
