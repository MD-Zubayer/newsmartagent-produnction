# Printing Order Labels

Speed up your shipping process by printing professional order labels directly from the NewsSmartAgent dashboard.

### 🖨️ Step-by-Step

1.  Go to **CRM > Orders**.
2.  Select the order(s) you wish to process.
3.  Click the **'Print Label'** icon.
4.  **Configuration**: Choose your paper size (e.g., A4 or 4x6 Thermal).
5.  **Print**: A PDF will be generated with the customer's name, phone, address, and Order ID.

### 📦 Batch Printing

You can select multiple orders and click **'Batch Print'** to generate a single PDF containing all labels, saving you significant time.

> [!TIP]
> Use a **Thermal Printer** for the best results and faster packaging.
