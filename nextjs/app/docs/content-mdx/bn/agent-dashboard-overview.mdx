# 📈 এজেন্ট ড্যাশবোর্ড: রেফারেল ও আর্নিং ওভারভিউ

New Smart Agent (NSA)-এর এজেন্ট ড্যাশবোর্ডটি এমনভাবে ডিজাইন করা হয়েছে যাতে একজন পার্টনার বা এজেন্ট তার রেফারেল নেটওয়ার্ক এবং উপার্জনের প্রতিটি তথ্য রিয়েল-টাইমে মনিটর করতে পারেন।

---

### 💰 ১. মূল ইনকাম ও ব্যালেন্স (Earnings & Wallet)

ড্যাশবোর্ডের উপরে আপনার গুরুত্বপূর্ণ আর্থিক তথ্যগুলো কার্ড আকারে প্রদর্শিত হয়:

*   **💳 Account Balance**: আপনার ওয়ালেটে থাকা মোট টাকার পরিমাণ (পেমেন্ট + কমিশন)।
*   **🎁 Commission Earned**: রেফারেল থেকে এখন পর্যন্ত মোট কত টাকা কমিশন আয় করেছেন তার হিসাব।

---

### 🔗 ২. রেফারেল নেটওয়ার্ক (Referral Insights)

আপনার নেটওয়ার্কের গ্রোথ ট্র্যাক করুন:

| মেট্রিক | বিবরণ |
| :--- | :--- |
| **👥 Total Referrals** | আপনার লিংকে ক্লিক করে মোট কতজন অ্যাকাউন্ট তৈরি করেছেন। |
| **✅ Active Referrals** | কতজন ইউজারবর্তমানে নিয়মিত সার্ভিসটি ব্যবহার করছেন। |

---

### 📊 ৩. সাবস্ক্রিপশন ডাটা (Subscription Tracking)

রেফার করা ইউজারদের অ্যাক্টিভিটি মনিটর করুন:

*   **📝 Total Subscriptions**: এখন পর্যন্ত মোট কতগুলো সাবস্ক্রিপশন সেল হয়েছে।
*   **🌟 Currently Active**: বর্তমানে কতজন ইউজারের অফার সচল আছে।
*   **⌛ Currently Inactive**: যেসব ইউজারের অফারের মেয়াদ শেষ এবং রিনিউ প্রয়োজন।

---

> [!TIP]
> **বেশি আয়ের টিপস**: আপনার 'Inactive' ইউজারদের সাথে যোগাযোগ করুন এবং তাদের নতুন অফার সম্পর্কে জানান। এতে আপনার কমিশন বাড়ার সম্ভাবনা বৃদ্ধি পাবে!

> [!NOTE]
> কমিশন সরাসরি আপনার ওয়ালেটে যোগ হয়, যা আপনি পরবর্তী কেনাকাটায় ব্যবহার করতে পারেন অথবা উইথড্র করার সুযোগ পেতে পারেন।
