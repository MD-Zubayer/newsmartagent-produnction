# অ্যাকাউন্ট ব্যালেন্স যুক্ত করা

আপনার অ্যাকাউন্ট ব্যালেন্স সাবস্ক্রিপশন এবং AI API ইন্টারঅ্যাকশনের জন্য ব্যবহৃত হয়। ব্যালেন্স ঠিক রাখা নিরবিচ্ছিন্ন সেবা নিশ্চিত করে।

### 💵 ডিপোজিট পদ্ধতিসমূহ

আমরা বাংলাদেশে বিস্তৃত নিরাপদ পেমেন্ট পদ্ধতি সমর্থন করি:
*   **বিকাশ / নগদ / রকেট**: তাৎক্ষণিক মোবাইল ফিন্যান্সিয়াল সার্ভিস ট্রান্সফার।
*   **ডেবিট/ক্রেডিট কার্ড**: ভিসা, মাস্টারকার্ড এবং ডিবিবিএল (DBBL)।
*   **ব্যাংক ট্রান্সফার**: বড় এন্টারপ্রাইজ ডিপোজিটের জন্য।

### 🚀 ব্যালেন্স টপ-আপ করার নিয়ম

১. **Billing > Add Balance**-এ যান।
২. বিডিটি (BDT) হিসেবে পরিমাণ লিখুন (ন্যূনতম ৫০০ টাকা)।
৩. **'Proceed to Pay'**-এ ক্লিক করুন এবং নিরাপদ গেটওয়েতে লেনদেন সম্পন্ন করুন।
৪. সফল পেমেন্টের পর আপনার ব্যালেন্স তাৎক্ষণিকভাবে আপডেট হবে।

> [!IMPORTANT]
> পিক আওয়ারে আপনার AI এজেন্ট যাতে রেসপন্স করা বন্ধ না করে তা নিশ্চিত করতে সর্বদা ন্যূনতম ২০০ টাকা ব্যালেন্স রাখুন।
