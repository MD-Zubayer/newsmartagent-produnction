# অর্ডার লেবেল প্রিন্ট করা

নিউজমার্টএজেন্ট (NewsSmartAgent) ড্যাশবোর্ড থেকে সরাসরি প্রফেশনাল অর্ডার লেবেল প্রিন্ট করে আপনার শিপিং প্রক্রিয়াকে আরও দ্রুত করুন।

### 🖨️ ধাপে ধাপে নির্দেশিকা

১. **CRM > Orders**-এ যান।
২. যে অর্ডার বা অর্ডারগুলোর লেবেল প্রিন্ট করতে চান তা সিলেক্ট করুন।
৩. **'Print Label'** আইকনে ক্লিক করুন।
৪. **কনফিগারেশন**: আপনার কাগজের সাইজ বেছে নিন (যেমন: A4 বা 4x6 থার্মাল)।
৫. **প্রিন্ট**: কাস্টমারের নাম, ফোন, ঠিকানা এবং অর্ডার আইডি-সহ একটি PDF তৈরি হবে।

### 📦 ব্যাচ প্রিন্টিং (Batch Printing)

আপনি একসাথে অনেকগুলো অর্ডার সিলেক্ট করে **'Batch Print'**-এ ক্লিক করতে পারেন। এতে করে সব লেবেল নিয়ে একটি মাত্র PDF তৈরি হবে, যা আপনার অনেক সময় বাঁচাবে।

> [!TIP]
> সেরা ফলাফল এবং দ্রুত প্যাকেজিংয়ের জন্য একটি **থামাল প্রিন্টার (Thermal Printer)** ব্যবহার করার পরামর্শ দেওয়া হচ্ছে।
