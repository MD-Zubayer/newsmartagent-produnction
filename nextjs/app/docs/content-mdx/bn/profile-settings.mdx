# প্রোফাইল ও ইউজার সেটিংস

আপনার নিউজমার্টএজেন্ট (NewsSmartAgent) অভিজ্ঞতাকে কাস্টমাইজ করুন এবং আপনার অ্যাকাউন্টের বিবরণ আপ-টু-ডেট রাখুন।

### 👤 প্রোফাইল ম্যানেজমেন্ট

*   **নাম পরিবর্তন**: প্ল্যাটফর্মে এবং রিপোর্টে প্রদর্শিত নাম আপডেট করুন।
*   **ফোন নম্বর পরিবর্তন**: নিরাপত্তার জন্য, আপনার ফোন নম্বর পরিবর্তন করতে ওটিপি (OTP) ভেরিফিকেশনের প্রয়োজন হবে।
*   **আভাটার/লোগো**: আপনার অ্যাকাউন্ট রিপ্রেজেন্ট করতে ব্যবসায়িক লোগো বা ব্যক্তিগত ছবি আপলোড করুন।

### ⚙️ প্রেফারেন্স সেটিংস

১. **নোটিফিকেশন সেন্টার**: নতুন লিড আসার পর ইমেইল বা প্ল্যাটফর্ম নোটিফিকেশনের মাধ্যমে অ্যালার্ট পেতে পছন্দ করুন।
২. **ভাষা**: আপনার ড্যাশবোর্ডের ডিফল্ট ভাষা (EN/BN) সেট করুন।
৩. **টাইমজোন**: সঠিক রিপোর্টিংয়ের জন্য আপনার স্থানীয় টাইমজোন অনুযায়ী অ্যাডজাস্ট করুন।

> [!TIP]
> আপনার **Business Name** সর্বদা আপডেট রাখুন, কারণ AI এজেন্ট গ্রাহকদের কাছে নিজের পরিচয় দেওয়ার সময় এটি ব্যবহার করে।
