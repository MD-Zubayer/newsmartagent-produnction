# 📝 অ্যাকাউন্ট রেজিস্ট্রেশন গাইড

আমাদের এআই অটোমেশন সেবা এবং ড্যাশবোর্ড অ্যাক্সেস করতে আপনাকে প্রথমে একটি অ্যাকাউন্ট তৈরি করতে হবে। নিরাপত্তার স্বার্থে শুধুমাত্র নিবন্ধিত ব্যবহারকারীরাই ড্যাশবোর্ড ব্যবহার করতে পারেন।

---

### 🛠️ পদ্ধতি ১: ম্যানুয়াল রেজিস্ট্রেশন (Manual Registration)

১. ওয়েবসাইট থেকে **"Get Started free"**, **"Sign In > Create an Account"** অথবা যেকোনো **"Create Account"** লিঙ্কে ক্লিক করুন।
২. মোবাইল ব্যবহারকারীরা **Menu (☰)** আইকনে ট্যাপ করে **Sign In > Create an Account** অপশনটি নির্বাচন করুন।

#### 📝 রেজিস্ট্রেশন ফর্ম পূরণ করুন:
*   **পুরো নাম (Full Name):** আপনার নাম বা ব্যবসার নাম লিখুন।
*   **যোগাযোগ:** একটি সচল ফোন নম্বর এবং ইমেল ঠিকানা দিন।
*   **ঠিকানা:** আপনার বিভাগ, জেলা এবং উপজেলা নির্বাচন করুন।
*   **অ্যাকাউন্ট খোলার মাধ্যম (Created By):** 
    *   নিজে অ্যাকাউন্ট খুললে **'Self'** সিলেক্ট করুন।
    *   অন্য কারো মাধ্যমে খুললে **'Agent'** সিলেক্ট করুন (এক্ষেত্রে এজেন্টের ইউনিক আইডি এবং ওটিপি কি প্রয়োজন হবে)।
*   **অ্যাকাউন্টের ধরন:** **User** অথবা **Agent** যেকোনো একটি বেছে নিন (একটি অ্যাকাউন্ট দিয়ে উভয় ড্যাশবোর্ডই ব্যবহার করা যাবে)।
*   **পাসওয়ার্ড:** বড় হাতের অক্ষর, সংখ্যা এবং চিহ্নের সমন্বয়ে একটি শক্তিশালী পাসওয়ার্ড তৈরি করুন।

৩. সবশেষে **"Register Account"** বাটনে ক্লিক করুন।

> [!IMPORTANT]
> **ইমেল ভেরিফিকেশন:** রেজিস্ট্রেশন করার পর আপনার ইমেলে একটি ভেরিফিকেশন লিঙ্ক পাঠানো হবে। সেখানে ক্লিক করলেই আপনার অ্যাকাউন্টটি সক্রিয় হবে।

---

### 🤖 পদ্ধতি ২: এআই বট রেজিস্ট্রেশন (অটোমেটেড)

১. স্ক্রিনে থাকা **Bot আইকনে** ক্লিক করুন।
২. একটি মেসেজ উইন্ডো ওপেন হবে। আমাদের এআই বট আপনাকে ধাপে ধাপে নির্দেশনা দেবে।
৩. বটের জিজ্ঞাসা অনুযায়ী সঠিক তথ্য প্রদান করুন।
৪. বট আপনার অ্যাকাউন্ট তৈরি করে দেওয়ার পর, আপনার ইমেল চেক করুন এবং ভেরিফিকেশন লিঙ্কে ক্লিক করে একাউন্ট চালু করুন।
