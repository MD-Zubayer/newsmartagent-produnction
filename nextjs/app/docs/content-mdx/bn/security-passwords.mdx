# পাসওয়ার্ড সংক্রান্ত সেরা অভ্যাস

আপনার নিউজমার্টএজেন্ট (NewsSmartAgent) অ্যাকাউন্ট এবং আপনার ব্যবসায়িক তথ্য নিরাপদ রাখতে এই পাসওয়ার্ড নির্দেশিকাগুলো অনুসরণ করুন।

### 🛡️ একটি শক্তিশালী পাসওয়ার্ড তৈরি করা

*   **দৈর্ঘ্য**: অন্তত ১২টি অক্ষর ব্যবহার করুন।
*   **জটিলতা**: বড় হাতের অক্ষর (`A-Z`), ছোট হাতের অক্ষর (`a-z`), সংখ্যা (`0-9`) এবং বিশেষ চিহ্নের (`!@#$%^&*`) সমন্বয় ব্যবহার করুন।
*   **অনন্যতা**: ফেসবুক বা গুগল অ্যাকাউন্টে ব্যবহৃত পাসওয়ার্ড এখানে পুনরায় ব্যবহার করবেন না।

### 🚫 যা এড়িয়ে চলবেন

১. **ব্যক্তিগত তথ্য**: আপনার নাম, ফোন নম্বর বা জন্ম তারিখ ব্যবহার করা এড়িয়ে চলুন।
২. **সাধারণ শব্দ**: "password123" বা "admin"-এর মতো শব্দ ব্যবহার করবেন না।
৩. **সহজ প্যাটার্ন**: "qwerty" বা "12345" এর মতো সহজ প্যাটার্নগুলো হ্যাকারদের জন্য খুব সহজ।

> [!TIP]
> একটি পাসফ্রেজ (Passphrase) ব্যবহার করুন—যেমন ৪-৫টি এলোমেলো শব্দের সংকেত (যেমন: "SkyBlueRunningWaterFast")—যা আপনার মনে রাখা সহজ কিন্তু হ্যাকারদের জন্য অনুমান করা কঠিন।
