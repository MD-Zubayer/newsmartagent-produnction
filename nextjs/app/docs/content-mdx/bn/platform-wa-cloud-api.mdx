# হোয়াটসঅ্যাপ ক্লাউড API বনাম লিঙ্কড ডিভাইস

নিউজমার্টএজেন্ট (NewsSmartAgent) ব্যবহারের সুবিধার জন্য মূলত **লিঙ্কড ডিভাইস (QR Scan)** পদ্ধতি ব্যবহার করে, তবে এন্টারপ্রাইজ-গ্রেড স্থায়িত্বের জন্য আমরা অফিসিয়াল **WhatsApp Cloud API**-ও সমর্থন করি।

### 🌓 তুলনা

| ফিচার | লিঙ্কড ডিভাইস (QR) | ক্লাউড API (অফিসিয়াল) |
| :--- | :--- | :--- |
| **সেটআপ** | ⚡ খুব সহজ | 🛠️ জটিল |
| **স্থায়িত্ব** | 📱 ফোন অনলাইন রাখা জরুরি | 🌐 সর্বদা অনলাইন |
| **খরচ** | 🆓 ফ্রি (শুধুমাত্র NSA) | 💳 প্রতি মেসেজ (Meta) |
| **ব্রডকাস্ট** | 🚫 সীমিত | ✅ আনলিমিটেড |

### 🛠️ ক্লাউড API-তে যাওয়ার নিয়ম

১. একটি **Meta Developer Account** তৈরি করুন।
২. আপনার **Permanent Token** এবং **Phone Number ID** সংগ্রহ করুন।
৩. এই তথ্যগুলো **Platforms > Advanced WhatsApp Settings**-এ দিন।

> [!WARNING]
> ক্লাউড API-তে যাওয়ার জন্য মেটা (Meta) থেকে পৃথক ব্যবসায়িক ভেরিফিকেশন প্রয়োজন। আপনার আইনি কাগজপত্র প্রস্তুত রাখুন।
