# সুরক্ষিত লগইন ধাপসমূহ

আপনার অ্যাকাউন্ট ভেরিফাই হয়ে গেলে, আপনি আপনার ড্যাশবোর্ডে লগইন করতে পারেন। আপনার ব্যবসার তথ্য সুরক্ষিত রাখতে আমরা ইন্ডাস্ট্রি-স্ট্যান্ডার্ড নিরাপত্তা ব্যবহার করি।

### 🔑 লগইন ওয়ার্কফ্লো

১. **গেটওয়েতে প্রবেশ**: লগইন পেজে যান।
২. **ইমেইল**: আপনার নিবন্ধিত ইমেইল ঠিকানাটি সাবধানে টাইপ করুন।
৩. **পাসওয়ার্ড**: আপনার পাসওয়ার্ডটি দিন। টাইপ করার সময় কোনো ভুল হয়েছে কিনা তা দেখতে **'আই আইকন' (Eye Icon)** ব্যবহার করুন।
৪. **লগইন বোতাম**: এগিয়ে যেতে 'Login' বাটনে ক্লিক করুন।

### 🛡️ সেরা নিরাপত্তা অভ্যাস

*   **প্রাইভেট ব্রাউজিং**: পাবলিক কম্পিউটারে পাসওয়ার্ড সেভ করা থেকে বিরত থাকুন।
*   **ইউনিক ক্রেডেনশিয়াল**: অন্য প্ল্যাটফর্মের পাসওয়ার্ড এখানে পুনরায় ব্যবহার করবেন না।
*   **সেশন টাইমআউট**: নিরাপত্তার জন্য, নিষ্ক্রিয় সেশনগুলো ২৪ ঘণ্টা পরে স্বয়ংক্রিয়ভাবে লগ আউট হয়ে যাবে।

> [!TIP]
> আপনার ক্রেডেনশিয়ালগুলো সুরক্ষিতভাবে সংরক্ষণ করতে এবং ভুলে যাওয়া এড়াতে পাসওয়ার্ড ম্যানেজার ব্যবহার করুন।
