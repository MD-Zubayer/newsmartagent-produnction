# পোর্টাল এক্সেস গাইড

NewsSmartAgent প্ল্যাটফর্ম ব্যবহার শুরু করার জন্য আপনাকে আমাদের অফিসিয়াল ওয়েব পোর্টালে প্রবেশ করতে হবে। নিরাপদ সংযোগ নিশ্চিত করতে নিচের ধাপগুলো অনুসরণ করুন।

### 🌐 পোর্টালে প্রবেশের ধাপসমূহ

1.  **ব্রাউজার ওপেন করুন**: সেরা অভিজ্ঞতার জন্য গুগল ক্রোম (Google Chrome), মজিলা ফায়ারফক্স (Mozilla Firefox), বা সাফারি (Safari)-এর মতো আধুনিক ব্রাউজার ব্যবহার করুন।
2.  **URL লিখুন**: অ্যাড্রেস বারে [**https://newsmartagent.com**](https://newsmartagent.com) টাইপ করুন।
3.  **নিরাপত্তা যাচাই করুন**: URL-এর পাশে থাকা প্যাডলক (প্যাডলক আইকন) দেখুন যাতে আপনার সংযোগটি এনক্রিপ্ট করা হয়েছে তা নিশ্চিত করা যায়।

### 📱 ডিভাইস অনুযায়ী ভিন্নতা

*   **ডেস্কটপ (Desktop)**: হোমপেজের উপরের ডানদিকের কোণায় 'Get Started' বোতামটি অবস্থিত।
*   **মোবাইল (Mobile)**: হ্যামবার্গার মেনু (তিনটি অনুভূমিক রেখা) ট্যাপ করুন এবং 'Sign Up' অথবা 'Login' নির্বাচন করুন।

> [!IMPORTANT]
> আপনার ব্যবসার ক্রেডেনশিয়াল সুরক্ষিত রাখতে সর্বদা অফিসিয়াল ডোমেনে আছেন কিনা তা নিশ্চিত করুন।
