# 🔐 পাসওয়ার্ড রিসেট করা

আপনি যদি পাসওয়ার্ড ভুলে যান, তবে চিন্তিত হবেন না। আমাদের সুরক্ষিত রিকভারি প্রোটোকল অনুসরণ করে আপনি পুনরায় নিউজমার্টএজেন্ট (NewsSmartAgent) অ্যাকাউন্টে প্রবেশ করতে পারেন।

---

### 🔄 রিকভারি ধাপসমূহ

১. **পাসওয়ার্ড ভুলে গেছেন?**: লগইন গেটওয়েতে, ক্রেডেনশিয়াল ফিল্ডের নিচে থাকা **'Forgot Password?'** লিঙ্কে ক্লিক করুন।
২. **ইমেইল দিন**: রিসেট লিংক পেতে আপনার নিবন্ধিত ইমেইল ঠিকানাটি প্রদান করুন।
৩. **ইনবক্স চেক করুন**: নিউজমার্টএজেন্ট থেকে পাঠানো রিকভারি ইমেইলটি খুলুন এবং **'Reset Password'** বাটনে ক্লিক করুন।
৪. **নতুন পাসওয়ার্ড তৈরি করুন**: একটি নতুন শক্তিশালী পাসওয়ার্ড দিন এবং সেটি কনফার্ম করুন।
৫. **পুনরায় লগইন করুন**: আপনার ড্যাশবোর্ডে প্রবেশ করতে নতুন পাসওয়ার্ড ব্যবহার করুন।

### 🛡️ নিরাপত্তা দ্রষ্টব্য

*   **সাময়িক লিংক**: আপনার নিরাপত্তার জন্য রিসেট লিংকটি মাত্র **৩০ মিনিট** কার্যকর থাকে।
*   **একবার ব্যবহারযোগ্য**: প্রতিটি লিংক মাত্র একবার ব্যবহার করা যাবে।
