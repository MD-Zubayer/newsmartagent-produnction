# সেশন টাইমআউট এবং নিরাপত্তা

আপনার সুরক্ষার জন্য, নিউজমার্টএজেন্ট (NewsSmartAgent) অননুমোদিত প্রবেশ রোধ করতে কঠোর সেশন ম্যানেজমেন্ট ব্যবস্থা বজায় রাখে, বিশেষ করে যদি আপনি লগ আউট করতে ভুলে যান।

### ⏳ এটি কীভাবে কাজ করে

*   **নিষ্ক্রিয়তা (Inactivity)**: আপনি যদি **২ ঘণ্টা** একটানা নিষ্ক্রিয় থাকেন, তবে আপনার সেশনটি স্বয়ংক্রিয়ভাবে শেষ হয়ে যাবে এবং আপনাকে পুনরায় লগইন করতে বলা হবে।
*   **হার্ড লিমিট**: আপনি সক্রিয় থাকলেও সেশনগুলো প্রতি **২৪ ঘণ্টা** অন্তর অন্তর রিফ্রেশ করা হয় যাতে সিকিউরিটি টোকেনগুলো আপ-টু-ডেট থাকে।

### 🛡️ ম্যানুয়াল লগআউট

আপনি যদি কোনো শেয়ার্ড কম্পিউটার বা পাবলিক ওয়াই-ফাই নেটওয়ার্ক ব্যবহার করেন, তবে সর্বদা সাইডবারের নিচে থাকা **'Logout'** বাটনে ক্লিক করুন।

> [!NOTE]
> সেশন টাইমআউট কোনো বাগ নয়, বরং একটি নিরাপত্তা ফিচার। আপনার ডিভাইসটি ভুল হাতে পড়লেও এটি আপনার ব্যবসায়িক লিডগুলোকে সুরক্ষিত রাখে।
