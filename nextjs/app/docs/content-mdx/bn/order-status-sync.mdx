# কুরিয়ার স্ট্যাটাস সিঙ্ক্রোনাইজেশন

নিউজমার্টএজেন্ট (NewsSmartAgent) বাংলাদেশের শীর্ষস্থানীয় কুরিয়ার সার্ভিসগুলোর সাথে ইন্টিগ্রেটেড, যা আপনার কাস্টমারদের রিয়েল-টাইম ট্র্যাকিং সুবিধা প্রদান করে।

### 🔗 কুরিয়ার কানেক্ট করা

১. **CRM > Courier Settings**-এ যান।
২. আপনার কুরিয়ার প্রোভাইডার বেছে নিন (যেমন: RedX, Pathao, Steadfast)।
৩. আপনার **API Key** বা **Sender ID** দিন।
৪. **সিঙ্ক (Sync)**: এখন থেকে কুরিয়ারের ডেলিভারি প্রগ্রেস অনুযায়ী আপনার অর্ডারের স্ট্যাটাস স্বয়ংক্রিয়ভাবে আপডেট হবে।

### 🤖 AI নোটিফিকেশন

কুরিয়ার যখন পার্সেলটি **'Out for Delivery'** হিসেবে চিহ্নিত করবে, তখন আপনার AI এজেন্ট স্বয়ংক্রিয়ভাবে কাস্টমারকে হোয়াটসঅ্যাপ বা মেসেঞ্জারে আপডেট পাঠাতে পারবে।

> [!IMPORTANT]
> আপনার API কী-গুলো সঠিক কি না তা নিশ্চিত করুন। যদি সিঙ্ক ব্যর্থ হয়, তবে সঠিক কারণ জানতে **Integration Logs** চেক করুন।
