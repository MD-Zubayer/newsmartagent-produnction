# API কী ম্যানেজমেন্ট

ডেভেলপার এবং অ্যাডভান্সড ইউজারদের জন্য নিউজমার্টএজেন্ট (NewsSmartAgent) সরাসরি API এক্সেস প্রদান করে, যার মাধ্যমে আমাদের AI ফিচারগুলো আপনার নিজস্ব এক্সটার্নাল অ্যাপ্লিকেশনে ইন্টিগ্রেট করতে পারেন।

### 🔑 কী (Key) তৈরি করা

১. **Settings > Developer Portal**-এ যান।
২. **'Generate API Key'**-এ ক্লিক করুন।
৩. কী-এর জন্য একটি নাম দিন (যেমন: 'My_External_App')।
৪. কী-টি কপি করে নিরাপদ স্থানে সংরক্ষণ করুন। নিরাপত্তার কারণে এটি পরে আর দেখানো হবে না।

### 🛡️ রোটেশন এবং রিভোক (Rotation and Revocation)

*   **রোটেশন**: যদি সন্দেহ হয় যে কী-টি ফাঁস হয়েছে, তবে পুরনো কী-টি বাতিল করে নতুন একটি তৈরি করুন।
*   **রিভোক (Revoke)**: এক্সটার্নাল এক্সেস বন্ধ করতে যেকোনো কী তাৎক্ষণিকভাবে নিষ্ক্রিয় করুন।

> [!CAUTION]
> আপনার API কী আপনার অ্যাকাউন্ট ডাটা এবং ব্যালেন্সের পূর্ণ এক্সেস দেয়। তাই এটি কখনও গিটহাবের (GitHub) মতো পাবলিক রিপোজিটরিতে শেয়ার করবেন না।
