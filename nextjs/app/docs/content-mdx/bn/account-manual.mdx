# অ্যাকাউন্ট খোলা (ম্যানুয়াল পদ্ধতি)

আপনি যদি এজেন্টের সাহায্য ছাড়াই নিজে নিজে রেজিস্ট্রেশন করতে চান, তবে ম্যানুয়াল ফর্মটি পূরণ করতে পারেন। এটি খুবই সহজ এবং এতে ২ মিনিটেরও কম সময় লাগে।

### ✍️ ফর্মের বিস্তারিত

১. **পুরো নাম**: আপনার নিজের বা আপনার ব্যবসার নাম লিখুন।
২. **ফোন নম্বর**: `+880` দিয়ে শুরু করে একটি সচল মোবাইল নম্বর দিন।
৩. **ইমেইল ঠিকানা**: আপনার সচল ইমেইল ব্যবহার করুন কারণ এটি ভেরিফিকেশনের জন্য ব্যবহৃত হবে।
৪. **অবস্থান**: ড্রপডাউন থেকে আপনার **বিভাগ**, **জেলা**, এবং **উপজেলা** নির্বাচন করুন।
৫. **Created By**: ড্রপডাউন মেনু থেকে **'Self'** নির্বাচন করুন।

### 🛡️ পাসওয়ার্ড সেট করুন

একটি শক্তিশালী পাসওয়ার্ড তৈরি করুন যাতে বড় হাতের অক্ষর, সংখ্যা এবং চিহ্ন থাকে। পাসওয়ার্ডটি টাইপ করার সময় কোনো ভুল নেই তা নিশ্চিত করতে পুনরায় টাইপ করুন।

> [!NOTE]
> ফর্মটি সাবমিট করার পরে, আপনার অ্যাকাউন্ট সক্রিয় করতে আপনি একটি ভেরিফিকেশন ইমেইল পাবেন।
