# সাবস্ক্রিপশন প্ল্যানসমূহ

নিউজমার্টএজেন্ট (NewsSmartAgent) স্টার্টআপ, ক্রমবর্ধমান ব্যবসা এবং বড় এন্টারপ্রাইজের জন্য উপযোগী বিভিন্ন ধরণের প্ল্যান অফার করে।

### 📦 প্ল্যান ক্যাটাগরিসমূহ

১. **Starter**: নতুন ব্যবসার জন্য সেরা। এতে ১টি প্ল্যাটফর্ম এবং ১টি AI এজেন্ট অন্তর্ভুক্ত।
২. **Professional**: ক্রমবর্ধমান ব্র্যান্ডের জন্য। এতে ৩টি প্ল্যাটফর্ম এবং উন্নত AI মডেল অন্তর্ভুক্ত।
৩. **Enterprise**: ডেডিকেটেড API সাপোর্ট এবং আনলিমিটেড এক্সেস সহ বড় ব্যবসার জন্য উপযোগী।

### 💳 কীভাবে সাবস্ক্রাইব করবেন

*   **Billing > Membership**-এ যান।
*   আপনার পছন্দসই প্ল্যানটি নির্বাচন করুন।
*   **'Subscribe'**-এ ক্লিক করুন। আপনার অ্যাকাউন্ট ব্যালেন্স থেকে ক্রেডিট কেটে নেওয়া হবে।

> [!TIP]
> যেকোনো প্রফেশনাল প্ল্যানে ২০% ডিসকাউন্ট পেতে **১২ মাসের** জন্য সাবস্ক্রাইব করুন।
